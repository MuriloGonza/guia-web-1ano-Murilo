O que é HTML, CSS & JS?

HTML

HTML é a linguagem de marcação que estrutura o conteúdo da web.

<h1>Olá Mundo!</h1>
<p>Este é um parágrafo.</p>

CSS

CSS é a linguagem que estiliza e formata o visual dos elementos HTML.

h1 {
    color: blue;
    font-size: 2rem;
}

JavaScript

JavaScript é a linguagem de programação que adiciona interatividade às páginas web.

document.querySelector('h1')
    .addEventListener('click', () => {
        alert('Olá!');
    });
